What we do

Architecture, design and implementation of web applications.

We create robust automated browser tests (webtests) that work.

We also automate build, deployment, tests and every other repeatable process.

We teach and couch developers on Ruby, Ruby on Rails and test automation.

Was hat die TU von mir bekommen?
